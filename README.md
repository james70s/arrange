# Kaoca-拷卡

媒体存储卡拷贝工具
Kaoca 拷卡是一款专门为视频创作者而设计的素材拷贝和备份工具，它可以方便的将你的素材安全拷贝到目标路径，并按照素材的年/月/日期的方式自动归档媒体文件。同时支持数据校验，让你拷贝数据高效无忧。

- 素材拷贝和备份
- 多路径同时备份
- 数据校验
- 自动整理
- 自动转码
- 短信通知
